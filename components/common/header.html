# header

用于处理头部特殊操作的组件

props: {
    backBtn: {
        type: Boolean,
        default: true
    },
    // 是否存在返回按钮
    title: {
        type: String
    },
    // 头部标签
    hasClickHandler: {
        type: Boolean,
        default: false
    },
    // 不传的话默认为返回上一次路由地址
},

<common-header title="到料单列表" :backBtn='false'>
    <div slot="left" @click="backToMaterial">
        <i class="nf-icon icon-back"></i>
    </div>
    <span slot='right' class="create-btn" @click="$router.push('/material-arrive-create')">新建</span>
</common-header>

像这种有特殊交互的就可以使用这个组件
